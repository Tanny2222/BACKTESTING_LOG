📖 วิธีแปลผล Monte Carlo
🎯 มันคืออะไร?
Monte Carlo สุ่มลำดับ Trade จากข้อมูล Backtest จริงของคุณซ้ำหลายพันรอบ — แต่ละรอบเรียงลำดับต่างกัน เพื่อดูว่า ระบบของคุณจะรอดหรือพังในสถานการณ์แย่สุด
📈 Equity Curve Fan
● เส้นเขียว (P95) = กรณีดีที่สุด 5% บน
● เส้นทอง (P50) = ผลลัพธ์กลาง ๆ ที่เกิดจริงบ่อยที่สุด
● เส้นแดง (P5) = กรณีแย่ที่สุด 5% ล่าง
Zone สีทอง = ช่วง P25–P75 ที่เป็นไปได้มากที่สุด
📉 Max Drawdown Histogram
แสดงว่า Max DD ที่เจอในแต่ละ simulation กระจุกอยู่ที่ระดับไหน
● แท่งสีแดง = simulation ที่ DD เกิน Ruin Threshold
ถ้าแท่งสีแดงมีเยอะ → Risk ต่อไม้ของคุณสูงเกินไป
🎰 Ruin Probability
% ของ simulation ที่ Port ลงลึกเกิน Ruin Threshold ที่ตั้งไว้

● < 3% = ปลอดภัย ✓
● 3–10% = ควรระวัง ⚠
● > 10% = Risk สูงเกินไป ✕
📊 Percentile Table
P50 = ผลลัพธ์ที่ "ปกติ" — ครึ่งหนึ่งของ simulation ได้มากกว่านี้ ครึ่งหนึ่งได้น้อยกว่า

P5 คือ "ถ้าโชคร้ายมาก" — ถ้า P5 ยัง Positive แสดงว่าระบบแข็งแกร่งมาก

ลด Risk % ต่อไม้ จนกว่า Ruin Probability จะ < 3%
